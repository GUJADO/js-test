Директивы DOCTYPE отвечает на определение типа документа, то есть сообщвет браузеру и валидатору, какая версия HTML исп. при написании страницы. В HTML5 версия в DOCTYPE не указывает что с точки зрения синтаксиса обес. для языка обратною совместность Допустим у вас есть сайт соотвествующий стандарту HTML 4.0 но вы хотите перевести его на HTML5 Для этого достаточно изменить DOCTYPE Все браузеры поддруживают сокращенное объявление DOCTYPE и отображает страницу в строгом соответствии стандартам
